# Kids Creator Toolkit

A lightweight, modern toolkit for building interactive branding and digital creation experiences for children and families. Built with Vite and TypeScript for fast performance, clean architecture, and easy customization.

---

## Features

- Polished, responsive UI built with modern web technologies
- Fast development workflow powered by Vite and TypeScript
- Modular structure designed for straightforward customization and extension
- Organized asset management for branding and visual projects

---

## Project Structure

```
kids-creator-toolkit/
├── index.html           # Main HTML entry point
├── src/
│   ├── assets/
│   │   └── images/      # Project images and branding assets
│   ├── App.tsx          # Root application component
│   └── index.css        # Global styles
├── vite.config.ts       # Vite configuration
├── tsconfig.json        # TypeScript compiler settings
└── package.json         # Dependencies and scripts
```

---

## Prerequisites

- Node.js 18 or later (LTS recommended)
- npm or yarn

---

## Getting Started

**1. Install dependencies**

```bash
npm install
```

**2. Start the development server**

```bash
npm run dev
```

Open the local address printed in the terminal to view the app.

**3. Build for production**

```bash
npm run build
```

---

## Development Workflow

1. Update components in `src/`
2. Modify global styles in `src/index.css`
3. Add or replace images in `src/assets/images/`
4. Preview changes with `npm run dev`
